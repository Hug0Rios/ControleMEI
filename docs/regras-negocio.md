# Regras de Negócio

## Limite MEI
- Limite anual: R$ 81.000
- Se empresa aberta no ano atual:
  - limite proporcional aos meses restantes

## Faturamento
- Soma de todas as receitas
- Pode vir de:
  - lançamento manual
  - importação de planilha
  - importação de notas fiscais

## Alertas
- 70% do limite → aviso
- 90% → alerta crítico
- 100% → limite atingido

## DAS
- Deve ser pago mensalmente
- Sistema apenas controla e alerta
- Emissão feita no portal oficial

## Notas fiscais
- XML deve ser lido e convertido em receita
- Valores devem bater com faturamento

## Usuário
- Cada usuário possui inicialmente 1 empresa
