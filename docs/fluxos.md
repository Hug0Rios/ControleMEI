# Fluxos

## Primeiro acesso
1. usuário cria conta
2. cadastra empresa
3. entra no dashboard

## Registrar receita
1. acessa faturamento
2. adiciona valor
3. sistema atualiza dashboard

## Importar nota
1. usuário faz upload XML
2. sistema lê dados
3. salva como receita

## Controle de DAS
1. sistema cria competência mensal
2. usuário marca como pago
