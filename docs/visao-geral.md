# Meu MEI Brasil

## Objetivo
Sistema completo de gestão para MEI, focado em:
- Controle de faturamento
- Limite anual (R$ 81.000)
- Controle de DAS
- Importação de notas fiscais
- Relatórios para contador

## Público alvo
- MEIs (prestadores de serviço e comércio)
- Contadores que atendem MEIs

## Problema resolvido
MEIs não têm controle do faturamento e acabam:
- Estourando limite
- Perdendo prazos do DAS
- Se desorganizando financeiramente

## Solução
Painel simples com:
- Dashboard financeiro
- Alertas automáticos
- Importação de dados
- Relatórios automatizados

## Diferenciais
- Foco total em MEI
- Simples e visual
- Integração com notas fiscais
- Relatórios prontos para contador
